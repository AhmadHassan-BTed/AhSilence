# Roadmap

This document outlines the planned evolution of AhSilence.

## Near-Term

- [ ] **Multi-frequency detection** — Cancel multiple overlapping hum sources simultaneously
- [ ] **Adaptive amplitude** — Auto-adjust anti-wave volume based on real-time feedback
- [ ] **Bluetooth latency compensation** — Automatic phase offset calibration for wireless audio

## Mid-Term

- [ ] **Frequency spectrum visualizer** — Real-time FFT magnitude display on the dashboard
- [ ] **Preset profiles** — Save and load calibration settings for different environments
- [ ] **Notification controls** — Quick-toggle phase/amplitude from the notification shade

## Long-Term

- [ ] **Machine learning frequency tracking** — Replace static FFT with adaptive models
- [ ] **Multi-device sync** — Coordinate cancellation across multiple phones/speakers
- [ ] **Wear OS companion** — Control AhSilence from a smartwatch
